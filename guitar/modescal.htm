Mode/Scale table

If you already know something about modes (for a really good explanation see a lesson by Dave Good at Harmony Central) this table may be of interest to you. As you will know a C Ionian scale , a D Dorain scale, a E Phrygian scale, aso. basically contain the same set of notes, namely the ones from the familiar C major(ionian) scale. Therefore you only need to know this one scale and automatically you know all the 7 modes of it, simply by starting at a different degree. So what Ionian scale do I have to use if I want to have a C Mixolydian ? That's where this table comes in: Just go down the Mixolydian column until you come to C and go back across to the left to find the corresponding Ionian scale.
